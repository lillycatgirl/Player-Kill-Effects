# Player Kill FX
This is a fully clientside fabric minecraft mod for 1.21.11 Java Edition that adds a laser beam whenever another player dies. No other mods are required except for a fabric installation.
Made as a joke for a friend and to get some more hours done.<br>
No AI was used in the creation of this mod, art or programming :3
